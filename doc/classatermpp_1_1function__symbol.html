atermpp::function_symbol::function_symbol ( )
inline

default constructor

A special function symbol constructor for use in the function symbol generator. This constructor assumes that the name and arity combination does not exist yet, and that the prefix and number combination is neatly recoreded in the appropriate number generator as being used. Furthermore, it takes a pointer to a char* to represent its string.
